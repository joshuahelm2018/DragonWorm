## This project demonstrates:

### CI/CD
• Run tests on commit/pull request and block builds when tests fail\
• Create build artifacts\
• Create downloadable zip release when version tag is committed

• Feature branching\
• Merging\
• Rebasing\
• Git LFS usage\
• Unity-friendly gitignore setup

### Unity
• Simple side-scrolling game\
• Uses Strategy Pattern\
• Basic menu system\
• TestRunner tests
